Information/ How to Use

  • I made this program to help myself practice scales on the guitar
  • Some things I'd like to add in the future are a more streamlined UI and native code

    • Keybinds:

  • "i": changes instrument
  • "r": toggles rainbow mode
  • "m" switches between Scale Mode and Mode Mode
  • left and right arrows change the root note
  • up and down arrows change the mode
  • right and left arrow keys to change the root position
