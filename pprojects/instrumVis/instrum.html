  • Some things I'd like to add in the future are a more streamlined UI and native code
    • How to Use

    How to Use

    (click on the canvas before using keybinds)

    Keybinds:

  • "i" - changes instrument
  • "r" - toggles rainbow mode
  • "m" - switches between Scales Mode and Modes Mode

    • When in Modes Mode, up and down arrow keys change the mode

    When in Scales Mode, "s" changes the scale

    Right and left arrow keys - move the root position
