Linear Algebra

I made this program to visualize the concepts I am learning in linear algebra

Keybinds

  • "p" - toogle smooth rotation
  • "r" - reset the matrix
  • "t" - transpose the matrix
  • up and down arrow keys - rotate about the X axis
  • ";" and "'" - rotate about the Y axis
  • left and right arrow keys - rotate about the Z axis
  • "," and "." - zoom in/out
