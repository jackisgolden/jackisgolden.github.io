I made this program to visualize the concepts I am learning in linear algebra
Some things I'd like to add in the future are a more streamlined UI and native code
Use the "r" key to reset the matrix
Use the left and right arrow keys to rotate about the Z axis
Use the up and down arrow keys to rotate about the x axis<
Use the ";" and "'" keys to rotate about the y axis<
Use the "," and "." keys to zoom
Use the "t" key to transpose the matrix
Use the "i" key to invert the matrix
